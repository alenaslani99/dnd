# DND

DND is a modern e-commerce project built to provide a clean, practical, and visually appealing online shopping experience. The platform is focused on presenting products in a simple and accessible way, while guiding users smoothly through the full purchase journey.

## Highlights

- 🛍️ Product-focused shopping experience
- 🧴 Brand and fragrance catalog structure
- 🛒 Cart and checkout flow
- 🔐 User authentication and profile access
- 📦 Order details and order tracking
- 📩 Contact form for customer communication
- 🛠️ Admin dashboard for products and order management

DND includes key customer-side features such as product pages, cart management, checkout, login and registration, profile access, contact functionality, and order tracking. On the administrative side, it also provides tools for managing products, monitoring orders, and updating order statuses, which makes the project suitable as a complete online store solution.
